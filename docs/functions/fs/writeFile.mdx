# fs.writeFile()

Overwrites or creates a new file with the contents.

## Usage:

```txt title="usage.ces"
import fs
fs.writeFile("monkey.txt", "Hello!") # file is created and content is "Hello!"
```

# Parameters:

The parameters of fs.writeFile(file, value)

## file*

The file to write to.

## value*

What to write in the file.

## Example

Same as usage.
