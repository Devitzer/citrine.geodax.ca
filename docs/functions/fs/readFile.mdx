# fs.readFile()

Reads a file from the directory that the file was ran from and returns the contents.

## Usage:

```txt title="usage.ob"
import fs
let file = fs.readFile("monkey.txt") # content: "Hello!"
io.print(file) # output: "Hello!"
```

# Parameters:

The parameters of fs.readFile(file)

## file*

The file to read.

Same as usage.
