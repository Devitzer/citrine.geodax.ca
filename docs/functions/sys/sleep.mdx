# sys.sleep()

Waits for the amount of milliseconds given.

# Usage

```txt title="usage.ob"
sys.sleep(2000)
io.print("Hello, World!") # prints after 2 seconds and this happens synchronously btw
```

# Parameters

The parameters of sys.sleep.

## milliseconds*

The amount of milliseconds to sleep for.
