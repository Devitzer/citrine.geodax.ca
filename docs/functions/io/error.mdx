# io.error()

Logs output to the console as an error. Supported types to log are: string | boolean | null | number

# Usage

```txt title="usage.ob"
io.error(...msg)
```

# Parameters

The parameters of io.error.

## msg

The type to log, can be several arguements.

# Example

```txt title="example.ob"
io.error("Hello, World! ", 123, true, null)
```
