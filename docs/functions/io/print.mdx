# io.print()

Logs output to the console. Supported types to log are: string | boolean | null | number

# Usage

```txt title="usage.ob"
io.print(...msg)
```

# Parameters

The parameters of io.print.

## msg

The type to log, can be several arguements.

# Example

```txt title="example.ob"
io.print("Hello, World! ", 123, true, null)
```
