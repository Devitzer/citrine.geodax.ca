# parse.json()

Parses a json file. Basically if you have a lets say a "test.json" file, and you want to parse it in CitrineScript. This is what that is for.

# Usage

```txt title="usage.ob"
static json = parse.json("test.json")
```

# Parameters

The parameters of parse.json.

## file

The file to read and parse.

# Example

```txt title="example.ob"
static json = parse.json("test.json")
```
