# parse.yaml()

Parses a YAML file. Basically if you have a lets say a "test.yaml|.yml" file, and you want to parse it in CitrineScript. This is what that is for.

# Usage

```txt title="usage.ob"
static yaml = parse.yaml("test.yml")
```

# Parameters

The parameters of parse.yaml.

## file

The file to read and parse.

# Example

```txt title="example.ob"
static yaml = parse.yaml("test.yml")
```
